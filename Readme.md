# HanuAI Assignment

## Overview
This project includes:
1. Web Scraping + Sentiment Analysis
2. EDA + Text Mining

---

## Tech Stack
- Python
- Pandas, NumPy
- Selenium, BeautifulSoup
- NLTK (VADER)
- Scikit-learn (TF-IDF, KMeans)
- Matplotlib, Seaborn

---

## How to Run

### Task 1:
cd Task1_Sentiment/src  
python scraper.py  
python sentiment.py  

### Task 2:
cd Task2-EDA/src  
python preprocessing.py  
python text_mining.py
